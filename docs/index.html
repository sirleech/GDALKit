Overview

Two modules ship in this package:

ImportPurpose
GDALKitThe Swift API documented here: GDALEnvironment, GDALRaster, CoordinateProjector.
CGDALThe raw GDAL/PROJ/SQLite C API, exposed from the xcframework via a module.modulemap. Import only for direct C access.

The wrapper is deliberately MapKit-free: geometry is expressed in Web Mercator (EPSG:3857) metres via MercatorBounds. Consuming apps map those to MapKit themselves.

import GDALKit

GDALEnvironment.bootstrap()                       // once, at launch
let raster = await GDALRaster.load(from: url)   // warp to 3857 off-main-thread

enumGDALEnvironment

One-time process setup for GDAL + PROJ. Call once before any other GDALKit API.

static func bootstrap()

Registers GDAL drivers and points PROJ/GDAL at the bundled proj.db + gdal data (resolved from Bundle.module). Idempotent and thread-safe. Mandatory — without it, datum/CRS transforms fail silently.

static func shareDirectory() -> URL?

Locates the bundled share/ directory (containing proj/proj.db and gdal/…). Rarely needed directly.

classGDALRaster

Wraps one georeferenced raster. On load it warps the source to EPSG:3857 once (tiled + overviewed temp GeoTIFF); subsequent reads are cheap axis-aligned windowed RasterIO reads. Reads are served from a thread-safe pool of dataset handles.

static func load(from sourceURL: URL, onProgress: (@Sendable (WarpPhase, Double) -> Void)? = nil) async -> GDALRaster?

Loads + warps on a background queue. Returns nil if the source can't be opened/warped. Accepts GeoTIFF (and any GDAL-readable georeferenced raster). Run bootstrap() first. The optional onProgress reports the current WarpPhase (.reprojecting / .buildingOverviews) and that phase's own 0…1 fraction — for a load progress bar. It fires frequently on the background load thread, so throttle and hop to the main actor before driving UI.

let bounds: MercatorBounds

Web-Mercator footprint of the warped raster, in metres.

func readImage(minX:maxX:minY:maxY:outW:outH:) -> (image: CGImage, bounds: MercatorBounds)?

Renders an arbitrary Web-Mercator window (metres) into an outW×outH RGBA CGImage. Returns the exact covered extent — the request snapped out to whole source pixels. Draw the image at the returned bounds, not the requested rect, so content registers precisely (see issue #16). Returns nil if fully outside the footprint.

if let t = raster.readImage(minX: x0, maxX: x1, minY: y0, maxY: y1, outW: 768, outH: 768) {
    // t.image : CGImage    t.bounds : MercatorBounds (place the image here)
}

func readTilePNG(x: Int, y: Int, z: Int) -> Data?

Renders one 256×256 PNG tile for the standard XYZ/Web-Mercator tile scheme.

static let mercatorMax: Double

Half the world width in EPSG:3857 metres (20037508.342789244); the ± extent of Web Mercator in both axes.

structMercatorBounds

An axis-aligned rectangle in EPSG:3857 (Web Mercator) metres. Equatable, Sendable.

struct MercatorBounds {
    var minX: Double; var minY: Double
    var maxX: Double; var maxY: Double
}

Convert to MapKit in the app, e.g. via the linear MKMapPoint↔EPSG:3857 relationship.

classCoordinateProjector

Transforms coordinates between two CRSs identified by EPSG code, over PROJ. Coordinates use traditional GIS axis order: (x, y) = (longitude/easting, latitude/northing).

Not thread-safe — create one per thread or serialise access. Requires bootstrap() first.

init?(fromEPSG: Int32, toEPSG: Int32)

Builds the transform. Returns nil if a CRS can't be constructed (usually a missing proj.db).

func project(x: Double, y: Double) -> (x: Double, y: Double)?

Projects a single point from source to destination CRS. nil if untransformable.

func project(xs: inout [Double], ys: inout [Double]) -> Bool

Bulk in-place transform of parallel coordinate arrays. Returns whether every point transformed.

let p = CoordinateProjector(fromEPSG: 4326, toEPSG: 3857)
let merc = p?.project(x: 151.2093, y: -33.8688)   // Sydney → ~(16_832_500, -4_011_500)

moduleCGDAL

The raw GDAL/PROJ/SQLite C API. Exposes gdal.h, cpl_*.h, gdal_utils.h, gdalwarper.h, and ogr_srs_api.h as the module CGDAL (a bridging header can't cross a package boundary). Import it only when you need GDAL/OGR/PROJ functions the GDALKit wrapper doesn't surface.

import CGDAL
GDALAllRegister()
let ds = GDALOpen(path, GA_ReadOnly)
Hand-written API reference. GDAL, PROJ, SQLite and the bundled proj.db/gdal data retain their own upstream licenses.
